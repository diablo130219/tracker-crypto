# CryptoNow Bankroll Tracker V8

Versione con branding coordinato, logo rotondo con cornice, sfondo più luminoso/contrastato e animazioni UI.

## Deploy Render
Build command:
```bash
pip install -r requirements.txt
```
Start command:
```bash
gunicorn app:app
```

## Variabili ambiente
- `APP_SECRET`
- `TRACKER_API_TOKEN`
- `STARTING_BANKROLL`
- `CURRENCY`
